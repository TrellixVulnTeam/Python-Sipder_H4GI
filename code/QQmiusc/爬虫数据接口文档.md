##### FlaskAPI接口

- request请求

  ```
  GET http://hostIP:port/data/top_id/[num]
  ```

- 请求参数

   ```json
   top_id string // 获取歌曲所属分类ID
      num string // 获取歌曲数量(不传递此参数默认全部数据)
   ```

- request响应

   正常响应 ： 

   ```json
     {
      'code' : "200"
      'data' ： [
                 {
                      'top_id' : string  // 歌曲所属分类ID
                      'songid' : string  // 歌曲ID
                      'songname' : string  // 歌曲名称
                      'songorig' : string  // 歌曲所属专辑
                   },
   				{
                      'top_id' : string  // 歌曲所属分类ID
                      'songid' : string  // 歌曲ID
                      'songname' : string  // 歌曲名称
                      'songorig' : string  // 歌曲所属专辑
      			  }
   			]
     }
   ```



  失败响应1：

  ```json
  {
      'code' : "0"
      'msg' : "数据异常,请稍后重试"
  }
  ```

失败响应2： 

```json
{
    'code': "700"
    'msg' : "请检查传入参数是否正确"
}
```
